# Pilotgram — como ter o guia sempre à vista no Cursor

Eu (o agente) **não consigo** abrir ou fixar separadores no teu IDE por ti.

### Fixar ao lado (recomendado)

1. No **Explorer**, abre `PILOTGRAM/COPY_PASTE_PG.md` com duplo clique.
2. **Arrasta** o separador desse ficheiro para a **direita** da janela até aparecer a zona de divisão → larga (editor em colunas).
3. Ou: com o ficheiro ativo, **`Cmd + \`** (macOS) / **`Ctrl + \`** (Windows) para dividir o editor; abre o mesmo ou outro ficheiro no novo painel.
4. Clica com o direito no separador → **Pin Tab** (fica sempre visível).
